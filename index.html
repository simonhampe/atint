a-tint

A polymake extension for tropical intersection theory

Download as .zip Download as .tar.gz View on GitHub

What is a-tint?

a-tint is an application for polymake (see their webpage for more info). It aims to implement algorithmic aspects of tropical intersection theory. Tropical varieties are realized as weighted polyhedral complexes. This project is part of and financed by the DFG Priority Project SPP 1489.

Getting started

You can find a user manual in the wiki pages, as well as installation instructions and a list of features.

Important note: You will need a recent version of polymake (perpetual beta, preferably) to run a-tint.
